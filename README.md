# jQuery Lite

jQuery Lite exposes a global function $l() that can be passed a string selector or a function.
It returns an instance of a DOMNodeCollection, which has many functions on its prototype, including:
- .html()
- .empty()
- .append()
- .attr()
- .addClass()
- .removeClass()
- .children()
- .parent()
- .find()
- .remove()
- .on()
- .off()
